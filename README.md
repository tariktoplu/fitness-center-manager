# Fitness Center Manager
**Fitness Center Manager** is a full-stack web application designed to streamline the management of gyms and fitness centers. The application allows gym owners and administrators to efficiently handle memberships, trainers, schedules, and payments.

## Features

- **Member Management:** Add, edit, and track gym members and their subscriptions.  
- **Trainer Management:** Assign trainers to members and manage schedules.  
- **Class Scheduling:** Create and manage classes, workouts, and events.  
- **Payment Tracking:** Record and monitor membership payments and invoices.  
- **Dashboard:** Overview of gym performance, active members, and upcoming events.

## Technologies

- **Backend:** ASP.NET / .NET Framework  
- **Frontend:** HTML, CSS, JavaScript, React  
- **Database:** SQL Server / PostgreSQL
